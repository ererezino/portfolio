# ererezino.com

Personal portfolio and blog by Zino Asamaige — co-founder building [Accrue](https://www.accrue.money) and [Cashramp](https://www.cashramp.co). Photographer, traveler, and walker based in Lagos, Nigeria.

**Live site:** [www.ererezino.com](https://www.ererezino.com)

---

## Structure

```
├── index.html              # Homepage
├── about.html              # About page
├── contact.html            # Contact form
├── contact-success.html    # Contact form confirmation
├── travels.html            # Travel map
├── movement.html           # Movement / walking page
├── photos/                 # Photography gallery
├── articles/               # Blog posts
│   ├── why-im-building-accrue/
│   ├── the-friction-tax/
│   ├── accrues-big-day/
│   ├── on-walking/
│   └── my-first-post/
├── assets/                 # Static assets
│   ├── photos/             # Photography images
│   ├── fonts/              # Self-hosted fonts
│   ├── leaflet/            # Map library
│   └── accrue-logos/       # Brand logos
├── docs/                   # Reference configs
│   ├── google-apps-script.js
│   └── SECURITY-HEADERS.txt
├── styles.css              # Global styles
├── script.js               # Main JavaScript
├── 404.html                # Custom error page
├── manifest.json           # PWA manifest
├── sitemap.xml             # Sitemap
├── robots.txt              # Crawler rules
└── feed.xml                # RSS feed
```

## Tech

Static HTML, CSS, and vanilla JavaScript. No frameworks or build tools. Hosted on Cloudflare Pages.
